Personal Website & Portfolio

Overview

This personal website serves as a professional online presence and a showcase for my projects and academic work. It's designed to present who I am, what I do, and what I aspire to achieve.

Purpose

Having a professional online presence is valuable for connecting with others who share similar interests. This website provides a central location where potential collaborators or anyone interested can learn about my background, projects, and academic focus.

What I Learned

Building this website from scratch provided valuable learning experiences:

  • Modern Web Development: Writing clean, semantic HTML and CSS
  • Responsive Design: Creating layouts that work on all devices from mobile to desktop
  • GitHub Pages: Setting up and deploying a static site with version control
  • Domain Configuration: DNS management and connecting a custom domain
  • Performance: Optimizing load times and user experience
  • Maintainability: Writing code that's easy to update and extend

Design Decisions

I chose to build a static site rather than using a website builder or CMS for several reasons:

  • Complete control over the design and functionality
  • No dependencies on external platforms
  • Fast loading times with minimal overhead
  • Free hosting on GitHub Pages
  • Learning experience in web development
